# vICE

Discord-inspired real-time communication platform built with modern web technologies.

## Overview

vICE is a real-time communication platform that enables users to create servers, join channels, exchange direct messages, and communicate through voice channels. The project focuses on delivering a modern user experience with low-latency communication and scalable backend architecture.

## Features

### Authentication & User Management

* User registration and login
* Profile management
* Friend system
* User presence tracking

### Real-Time Communication

* Real-time messaging with WebSocket
* Direct Messages (DM)
* Server-based communities
* Channel management
* Typing indicators

### Voice Communication

* Voice channels powered by LiveKit
* Voice presence synchronization
* Screen sharing support
* Adjustable screen share quality settings

### Moderation & Security

* Role and permission management
* Audit logging
* User blocking system
* Secure authentication flow

## Architecture

Frontend:

* React
* TypeScript
* Vite

Backend:

* Fastify
* TypeScript
* WebSocket

Communication:

* LiveKit
* Real-time Presence System

Database:

* PostgreSQL
